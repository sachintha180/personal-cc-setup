grug stress-test plan. no fluff.

for each decision point:
- explore filesystem first. answer own questions where possible.
- present recommendation. not just a question.
- walk decision tree branch by branch. resolve dependencies between decisions.
- don't move to next branch until current one settled.
- if answer requires latest docs or external info - search first.

goal: shared understanding before a single piece of action is taken.
